This is the project 1 for the Big Data Analytics Course (CIIC 5995-116), Spring 2017 at the University of Puerto Rico, Mayaguez Campus. MapReduce programs were designed, implemented, and tested to analyze a collection of tweets that included the word "Trump". Visualizations were made using the D3.js library.

Visualizations

  • Keywords
  • Word Frequency
  • Set of unique screen names that tweet
  • Retweets
  • Replies
  • Count of tweets by user
